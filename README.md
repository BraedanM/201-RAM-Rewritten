MAIN MENU  
The only features working on the main menu are the NEW GAME button and STORY button (EXIT as well but that doesnt work on unity editor)

CONTROLS  
WASD = Movement  
MOUSE POSITION = Player facing  
LEFT MOUSE CLICK = Shoot  
E = SPECIAL ATTACK (5s)
